# GamesForCats

GamesForCats is a playful mobile app built with Expo and React Native. It includes interactive mini-games and animated critters designed for cats, with customizable movement, speed, visuals, sounds, and emoji styles.

The app supports iOS and Android, uses file-based routing with Expo Router, and includes integrations for analytics and subscriptions.

## Tech Stack

- **Framework:** Expo + React Native (`expo` 54, `react-native` 0.81, `react` 19)
- **Language:** TypeScript
- **Routing & Navigation:** Expo Router + React Navigation
- **Animation & UI:** Reanimated, Gesture Handler, SVG, Glass Effect, Mesh Gradient
- **Audio & Haptics:** Expo Audio + Expo Haptics
- **Monetization & Analytics:** RevenueCat (`react-native-purchases`) + PostHog
- **Platforms:** iOS and Android (with web support via Expo)
