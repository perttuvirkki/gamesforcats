# App Store metadata (Fastlane deliver)

This folder contains per-locale App Store Connect metadata in the format expected by `fastlane deliver`.

Locales included:
- `en-US`
- `de-DE`
- `fr-FR`
- `es-ES`
- `fi`
- `sv`

Common files per locale:
- `name.txt`
- `subtitle.txt`
- `description.txt`
- `keywords.txt`
- `promotional_text.txt`
- `release_notes.txt`
- `support_url.txt`
- `marketing_url.txt`
- `privacy_url.txt`

Edit the text files as needed before running deliver.
